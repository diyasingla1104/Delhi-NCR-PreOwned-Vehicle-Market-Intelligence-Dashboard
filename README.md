# 📊 Delhi NCR Pre-Owned Vehicle Market Intelligence Dashboard

## 🔹 Project Overview

This project analyzes the **pre-owned vehicle market in Delhi NCR** using Power BI. It provides meaningful insights into pricing trends, popular brands, fuel types, and overall market demand patterns.

## 🔹 Tools Used

* Microsoft Power BI
* Excel / CSV Dataset
* Data Cleaning & Data Visualization

## 🔹 Key Insights

* 🚗 Most popular car brands in Delhi NCR
* 💰 Price distribution of used vehicles
* ⛽ Fuel type analysis (Petrol / Diesel / Electric)
* 📍 Location-wise vehicle availability
* 📅 Year-wise trends in the used car market

## 🔹 Dashboard Features

* Interactive filters (Price, Brand, Fuel Type)
* Dynamic charts and KPIs
* Clean and user-friendly design
* Easy navigation for decision-making
